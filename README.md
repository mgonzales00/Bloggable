# Bloggable
Simple blog site using Flask, Flask-Login, Flask-WTF/WTForms, Flask-CKEditor, Argon2, and SQLAlchemy.

### Features
- Secure login 🔐
- Text and image support 🖼️💬
- Create and edit posts ✍️📝
- Beautiful theming with Bootstrap 🎨✨
- Animated notifications 🛎️💥
